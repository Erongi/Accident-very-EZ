# I see dead people
# Description
วิเคราะห์การเกิดอุบัติเหตุการจราจรทางบกในด้านสาเหตุที่ทำให้เกิดอุบัติเหตุและด้านประเภทรถและความเสียหายทั่วราชอณาจักร ตั้งแต่ปี 2550-2557
